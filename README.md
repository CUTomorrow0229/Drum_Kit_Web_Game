# Drum Kit Web Game

為了練習JavaScript的Event listener, 動畫效果，用JavaScript, HTML, CSS製作而成的網頁遊戲  
目前主要有鍵盤及滑鼠的互動，按鈕的特效等等
遊戲網址：[Link](https://para0229k.github.io/Drum_Kit_Web_Game/)

## 1. 檔案結構

```
Drum_Kit_Web_Game/
├── images/       # 影像
│   ├── crash.png 
│   ├── kick.png  
│   ├── snare.png 
│   ├── tom1.png  
│   ├── tom2.png  
│   ├── tom3.png  
│   └── tom4.png  
├── sounds/       # 音效
│   ├── crash.mp3 
│   ├── kick-bass.mp3
│   ├── snare.mp3 
│   ├── tom-1.mp3 
│   ├── tom-2.mp3 
│   ├── tom-3.mp3 
│   ├── tom-4.mp3 
├── index.html    # 網頁素材配置
├── index.js      # 事件互動邏輯
└── styles.css    # 素材外觀設定

```

---

## 2. 功能介紹

|功能|說明|
|---|---|
|滑鼠互動|可用滑鼠點擊觸發按鈕|
|鍵盤互動|按下對應的按鍵也可以觸發按鈕|
|按鈕特效|按鈕觸發的時候會有視覺特效及音效|
